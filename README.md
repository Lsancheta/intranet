# Intranet PDP – Sistema de Gestão Interna

Este é um sistema de intranet desenvolvido para a empresa Pannela De Pedra, criado para otimizar processos internos como Ordens de Serviço, Estoque e Cozinha.  
O objetivo é digitalizar fluxos internos, reduzir erros manuais e melhorar a comunicação entre equipes.

---

## Tecnologias Utilizadas

- **Laravel** (Backend)
- **Vue.js + Inertia.js** (Frontend)
- **SQLite** (Banco de Dados)
- **TailwindCSS** (Estilos)
- **Tailscale** (Acesso remoto seguro em produção)

---

## Módulos do Sistema

###  Autenticação e Permissões
- Login / Logout
- Sistema de setores
- Controle de acesso baseado no setor do usuário

###  Ordens de Serviço (OS)
- Criação de OS
- Upload de fotos
- Edição, visualização e histórico
- Relacionamento com solicitantes

###  Estoque
- Cadastro de itens
- Registro de entrada e saída
- Controle básico de inventário

###  Administração
- Gestão de usuários
- Setores e permissões
